# German Vocabulary Extractor
Extract german vocabulary from text

![website demo](dump/images/website_demo.PNG)



## Run scripts

* Install packages.

```bash
pip install -r requirements.txt
```

* Run the streamlit app.

```bash
streamlit run lang_app.py 
```
